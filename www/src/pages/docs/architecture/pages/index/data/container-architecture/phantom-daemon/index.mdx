***phantom-daemon***, sistemin çekirdeğidir. WireGuard arayüzlerini, istemci yönetimini, firewall kurallarını ve veritabanlarını yönetir. Yönetim erişimi yalnızca UDS üzerinden sağlanır.

<Mermaid chart={`flowchart
    D[phantom-daemon] -- volumes --> PHANTOM_DB["./container-data/db\\n→ /var/lib/phantom/db"]
    PHANTOM_DB --> DB1[(Wallet)]
    PHANTOM_DB --> DB_exit[(Exit Store)]

    D -- volumes --> PHANTOM_STATE["./container-data/state/db\\n→ /var/lib/phantom/state/db"]
    PHANTOM_STATE --> DB2a[("WG Device State")]
    PHANTOM_STATE --> DB2b[(Firewall State)]

    D -- secrets --> WG_KEYS[WireGuard Keys]
`} />

## Çevre Değişkenleri

| Değişken                | Varsayılan                |
|-------------------------|---------------------------|
| `PHANTOM_DB_DIR`        | /var/lib/phantom/db       |
| `PHANTOM_STATE_DIR`     | /var/lib/phantom/state/db |
| `WIREGUARD_LISTEN_PORT` | 51820                     |
| `WIREGUARD_MTU`         | 1420                      |
| `WIREGUARD_KEEPALIVE`   | 25                        |
| `WIREGUARD_ENDPOINT_V4` | —                         |
| `WIREGUARD_ENDPOINT_V6` | —                         |

## Kernel Gereksinimleri

| Parametre               | Açıklama                                    |
|-------------------------|---------------------------------------------|
| `NET_ADMIN`             | WireGuard arayüz ve nftables kural yönetimi |
| `NET_RAW`               | Raw socket erişimi                          |
| `net.ipv4.ip_forward=1` | IP paket yönlendirme                        |
| `/dev/net/tun`          | TUN arayüz oluşturma                        |
