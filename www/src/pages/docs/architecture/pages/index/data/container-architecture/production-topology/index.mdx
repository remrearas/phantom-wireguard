***Üretim Topolojisi***, Docker Compose üzerinden yürütülen üç konteyner yapısından oluşur. Organizasyon üzerindeki yapının temel
görevi sorguyu güvenli bir şekilde ***Daemon*** yapısına aktarmaktır.

<Mermaid chart={`flowchart
    A[Internet] -->|"HTTPS :443 TLS"| B[phantom-nginx]
    B -->|"/auth/*  /api/*"| C[phantom-auth]
    C <-->|"UNIX socket\\ndaemon.sock"| D[phantom-daemon]

    A -->|"WireGuard\\nUDP 51820"| D
`} />

***Daemon***, yapısına ulaşan iki farklı trafik yolu vardır.

<Mermaid chart={`flowchart LR
  A[":443 HTTPS"] --> N["nginx"] --> AS["auth-service"] --> D["phantom-daemon"]
  B[":51820 UDP"] --> D
`} />


**Yönetim Trafiği (Kullanıcı Etkileşimi)** (`:443`): İnternet → phantom-nginx → phantom-auth → phantom-daemon

**VPN trafiği** (`:51820/udp`): WireGuard istemcileri doğrudan phantom-daemon'a bağlanır.

UDS soketi (`daemon.sock`) Docker named volume üzerinden phantom-auth ile paylaşılır.
