# Yedekleme ve Geri Yükleme

***Phantom-WG***, kullanıcı tarafından taşınabilir değerleri içeren veritabanlarını yedekleme imkanı sunar ve taşınabilir `.tar` arşivine aktarır. Arşiv üç dosyadan oluşur:

```text
phantom-backup-{timestamp}.tar
  ├── wallet.db         (istemci havuzu, temel yapılandırmalar)
  ├── exit.db           (exit tünel yapılandırmaları, multihop)
  └── manifest.json     (metadata: sürüm, zaman damgası, özet)
```

---

## Manifest

```json
{
  "version": "1.0",
  "timestamp": "2026-03-18T10:23:45.123456+00:00",
  "wallet": {
    "clients": 12,
    "subnet": "10.8.0.0/24",
    "pool_total": 253,
    "pool_assigned": 12
  },
  "exit_store": {
    "exits": 2,
    "enabled": false,
    "active_exit": ""
  }
}
```

## Doğrulama Katmanları

Yedekleme dosyası daemon'a ulaşmadan önce SPA tarafında, ardından daemon tarafında olmak üzere iki aşamalı doğrulamadan geçer:

### Tarayıcı Tarafı Doğrulama (SPA)

| Kontrol    | Kural                                          |
|------------|------------------------------------------------|
| Uzantı     | `.tar`                                         |
| Boyut      | Maks. 64 MB                                    |
| Tar yapısı | `wallet.db`, `exit.db`, `manifest.json` mevcut |
| Manifest   | Geçerli JSON, `version: "1.0"`                 |

### Sunucu Tarafı Doğrulama (Daemon)

| Kontrol                | Kural                                                                     |
|------------------------|---------------------------------------------------------------------------|
| Tar whitelist          | Yalnızca `wallet.db`, `exit.db`, `manifest.json` — fazla dosya reddedilir |
| Path traversal         | `../../` gibi yollar engellenir                                           |
| Manifest sürümü        | Geçersiz versiyon reddedilir                                              |
| PRAGMA integrity_check | Her iki veritabanı için bütünlük kontrolü                                 |
| Şema doğrulama         | Gerekli tablolar ve sütunlar kontrol edilir                               |
