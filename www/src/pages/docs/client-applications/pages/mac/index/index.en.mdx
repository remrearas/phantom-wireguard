## System Requirements

- macOS 14 (Sonoma) or later
- Apple Silicon (arm64) or Intel (x86_64)

---

## Download

### Homebrew (recommended)

```bash
brew install --cask ARAS-Workspace/tap/phantom-wg-mac
```

The cask places the signed `.app` into `/Applications`. The three system extensions are activated on first launch through the macOS approval flow.

### Direct Download

`.dmg` files are available from the [mac-v1.3.0 release](https://github.com/ARAS-Workspace/phantom-wg/releases/tag/mac-v1.3.0) page:

- `Phantom-WG-MacOS-1.3.0-arm64.dmg` — Apple Silicon
- `Phantom-WG-MacOS-1.3.0-x86_64.dmg` — Intel

Release build, signing, and notarization steps run on GitHub Actions via [mac-release.yml](https://github.com/ARAS-Workspace/phantom-wg/actions/workflows/mac-release.yml).
