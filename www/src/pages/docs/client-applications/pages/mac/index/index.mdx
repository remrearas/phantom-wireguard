## Sistem Gereksinimleri

- macOS 14 (Sonoma) veya üzeri
- Apple Silicon (arm64) veya Intel (x86_64)

---

## İndirme

### Homebrew (önerilen)

```bash
brew install --cask ARAS-Workspace/tap/phantom-wg-mac
```

Cask, imzalı `.app` dosyasını `/Applications` altına yerleştirir. Üç sistem uzantısı ilk açılışta macOS izin akışı üzerinden etkinleştirilir.

### Doğrudan İndirme

`.dmg` dosyalarına [mac-v1.3.0 release](https://github.com/ARAS-Workspace/phantom-wg/releases/tag/mac-v1.3.0) sayfasından erişilebilir:

- `Phantom-WG-MacOS-1.3.0-arm64.dmg` — Apple Silicon
- `Phantom-WG-MacOS-1.3.0-x86_64.dmg` — Intel

Sürüm derlemesi, imzalama ve notarizasyon adımları GitHub Actions üzerinde [mac-release.yml](https://github.com/ARAS-Workspace/phantom-wg/actions/workflows/mac-release.yml) ile yürütülür.
